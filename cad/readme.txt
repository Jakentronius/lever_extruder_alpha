This is a readme file intended to explain the extruder and its components contained in this directory.

The extruder is inteded to be printed out of whatever material the user wants and uses two drive elements to pinch the wire. 
One element is a passive 608RS bearing, link below:
https://www.mcmaster.com/products/bearings/bearing-trade-number~608-2rs/

The other active element is a large thumbscrew that is attached to a nema-17 motor via a printed sleve and set screw hole drilled into the side of the thumbscrew. Link below:
https://www.mcmaster.com/products/90368a350/

The wire is guided out of the pinched region and towards the printhead via an m6 push-connect pnumatic fitting. Link below:
https://www.mcmaster.com/7880T393
